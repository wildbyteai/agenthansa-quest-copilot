# State Machine

Allowed states:

- `FETCHING_QUEST_DETAIL`
- `ANALYZING_REQUIREMENTS`
- `WAITING_FOR_INFO`
- `PLANNING`
- `EVIDENCE_AUDIT`
- `CREATING_DELIVERABLE`
- `DELIVERABLE_REVIEW`
- `COMPLIANCE_CHECK`
- `READY_FOR_SUBMISSION_MATERIAL`
- `GRADE_HANDLING`
- `BLOCKED`

Forbidden states:

- `SUBMITTING`
- `SUBMITTED`
